Зліва: картина з плейсхолдером
Праворуч: 
- поле для вводу тексту
- під ним - картинки зі стилями і назвами
- кнопка Generate, що: 
    - викликає функцію
    - ховає форму і кнопку
    - показує текст запиту і вибраний стиль
    - показує нижче напис Generating...

коли отримуємо promise:
- зникає Generating...
- показуємо кнопку Download
- показуємо кнопку Generate new, яка повертає форму або перезавантажує сторінку
